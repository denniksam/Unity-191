User Interface
Интерфейс пользователя представляет собой 2D объект, содержащий текст, изображения,
элементы управления (кнопки, бегунки, ...)
При создании первого элемента UI дополнительно создаются
- Canvas - холст для размещения UI элементов (в 3D это внедрение 2D объектов)
- EventSystem - элемент, отвечающий за передачу событий UI в Engine

Настройка Canvas
= Render Mode      - режим совмещения холста и игры
 - Screen Space    - привязка к "экрану", нельзя менять размеры
  * Overlay        - привязка ко всем камерам - холст виден всегда
  * Camera         - привязка к одной из камер - холст всега виден только через одну камеру
 - World Space     - режим GameObject, можно сравнить с рекламным щитом

Выбираем Screen Space - Camera
 при этом появлется выбор камеры, к которой будет привязка - выбираем Main Camera
 Plane Distance - расстояние от камеры до плоскости холста

Часы и ход времени в Unity
Вызов методов UnityEngine происходит по сложной схеме при которой моменты вызова
методов не являются строго регламентированными
Для учета времени используется класс Time в котором свойство deltaTime отвечает
за интервал времени (в секундах), прошедший от предыдущего цикла. Общее время
можно узнать накапливая (складывая) все интервалы.
